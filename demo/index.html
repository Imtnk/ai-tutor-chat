AI Tutor Chat — Demo

Click the ✦ button in the bottom-right corner to open the tutor panel.

This demo uses the example-cc subject profile (ISC² CC exam tutor). Open Settings (⚙️) inside the panel to pick a provider:

  • Gemini — paste a free-tier API key from ai.google.dev.
  • Local — point at a local OpenAI-compatible server (e.g. Ollama with OLLAMA_ORIGINS="*" ollama serve). Enables web search via the bundled search bridge (see ../search-bridge/). Note: the model only gets a web_search tool — it has no separate clock/time tool, so it doesn't inherently know the current date or time unless a search result happens to reveal it.
  • Custom — any OpenAI-compatible Base URL.

Must be served over http://localhost, not file:// — local LLM servers block file:// origins via CORS.

AI role

Change what the tutor is scoped to. This only changes the assistant's role/behavior — it's separate from the provider settings (⚙️) inside the chat panel.

Note: this only changes the system prompt — the underlying model's own safety guardrails still apply on top of it. It will still refuse disallowed requests (e.g. explicit content) no matter what the role says.
