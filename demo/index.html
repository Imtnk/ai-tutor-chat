AI Tutor Chat — Demo

Click the ✦ button in the bottom-right corner to open the tutor panel.

This demo uses the example-cc subject profile (ISC² CC exam tutor). Open Settings (⚙️) inside the panel to pick a provider:

  • Gemini — paste a free-tier API key from ai.google.dev.
  • Local — point at a local OpenAI-compatible server (e.g. Ollama with OLLAMA_ORIGINS="*" ollama serve). Enables web search via the bundled search bridge (see ../search-bridge/).
  • Custom — any OpenAI-compatible Base URL.

Must be served over http://localhost, not file:// — local LLM servers block file:// origins via CORS.
